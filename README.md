# ember-node-demo #

一个简单的「Ember.js」和「Node.js」结合的CURD小应用，数据库使用「MongoDB」。

重点在于怎么样使用「Ember.js」进行前端开发，Enjoy It。

## 部署 ##

### 克隆模板到本地 ###

```
git clone https://github.com/happen-zhang/ember-node-demo.git ember-node
```

### 安装 ###

```
cd ember-node

npm install
```
